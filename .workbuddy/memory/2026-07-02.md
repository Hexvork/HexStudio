### 任务
用户要求把 LMMS 软件 UI 改得更新、符合 Windows 美学，并汉化（重点是乐器二级页面）。

### 决策（用户确认）
- 风格：Win11 Fluent 深浅双模式
- 深度：主题层为主（新建主题目录改 CSS，不动 C++）

### 完成内容
1. **Fluent 深色主题** `data/themes/fluent-dark/`

   - Segoe UI Variable 字体、#60CDFF 强调蓝、扁平按钮、圆角提升、干净滚动条、去拟物绿条
   - 调色板 qproperty 同步 Fluent token，PianoRoll 音符保留绿色语义

2. **Fluent 浅色主题** `data/themes/fluent-light/`
   - #0078D4 强调蓝、白底卡片层、浅色文字、配套配色

3. **中文汉化** `data/locale/zh_CN.ts`
   - 从 59.5%（1790/3007）补全到 100%（3007/3007）
   - 翻译 1217 条未译，覆盖 135 个 context
   - 重点是乐器/效果器二级页面：LOMM/Compressor/Microtuner/SlewDistortion/FreeBoy/Nes/Opulenz/Watsyn/Vibed/Xpressive/Monstro/Sf2 等

### 关键技术点
- 用 Python ET 解析/写回 .ts（保留 DOCTYPE），分 4 批应用翻译字典
- 主题纯新增目录，零破坏，可随时切回 default
- .qm 二进制由 cmake 构建 `finalize-locales` target 自动生成（系统未装 lrelease）

### 交付物
- `data/themes/fluent-dark/style.css` + 资源
- `data/themes/fluent-light/style.css` + 资源
- `data/locale/zh_CN.ts`（100% 翻译）

### 应用方式

- 翻译：构建后语言选 Chinese，或 `[app] language=zh`
